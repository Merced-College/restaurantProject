# 🍽️ Restaurant Review System

**Course:** CPSC 39 – Data Structures and Algorithms
**Assignment:** Group Programming Lab – Object-Oriented Programming with Java

---

# Overview

In this project, you will work with your assigned group to design and implement a restaurant review system using Java.

The program reads restaurant information from a CSV file and stores the data using nested objects and `ArrayList`s.

This assignment is designed to provide practice with:

* Object-Oriented Programming
* Java Classes and Objects
* Constructors
* Getters and Setters
* ArrayLists
* Nested Objects
* File Input
* Algorithms
* Team Collaboration
* GitHub Classroom

---

# Learning Objectives

By completing this assignment, you should be able to:

* Design Java classes from a UML diagram.
* Create and use objects.
* Store objects inside other objects.
* Read and process data from a file.
* Write algorithms that analyze collections of objects.
* Practice collaborative software development.
